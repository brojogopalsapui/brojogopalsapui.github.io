# Brojogopal Sapui Multi-Profile Website

This repository is organized as a single-site structure for a custom domain.

## Structure

- `/` — root landing page
- `/academic/` — classic academic profile
- `/ai-security/` — technical AI security portal

## Deployment

Upload everything to the root of your GitHub Pages repository (for example `brojogopalsapui.github.io`).

If your custom domain is already configured in GitHub Pages, the paths will become:

- `https://www.brojogopalsapui.com/`
- `https://www.brojogopalsapui.com/academic/`
- `https://www.brojogopalsapui.com/ai-security/`

## What to update first

1. Add your email, LinkedIn, Google Scholar, and ORCID on `academic/contact.html` and `ai-security/contact.html`
2. Expand `academic/publications.html` with your formal publication list
3. Update `academic/education.html` with detailed degree titles and dates if desired
4. Replace or add images in `ai-security/assets/img/` if you want new visuals
