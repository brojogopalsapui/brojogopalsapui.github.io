This file is for internal future use only.
Do not link this file directly from the website.

====================================================
HOW TO ADD A NEW WEEKLY / MONTHLY POST
====================================================

1. Copy the HTML template below
2. Create a new file inside:
   posts/
3. Use a name like:
   posts/2026-03-your-topic.html
4. Paste the HTML into that new file
5. Edit:
   - title
   - date
   - heading
   - introduction
   - overview text
   - discussion text
   - image path
   - related links
6. Open ongoing-work.html
7. Copy the accordion template already commented there
8. Paste it ABOVE the first existing watch-note
9. Add the new post link inside watch-inline-links

====================================================
BEST FILE NAMING STYLE
====================================================

posts/2026-03-secure-edge-agents.html
posts/2026-03-robotics-trust-boundary.html
posts/2026-04-llm-runtime-observation.html

Use:
- lowercase only
- hyphens only
- year-month-topic format
